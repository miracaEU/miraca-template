## Useful markdown tricks

Jupyter book allows for some special formatted boxes e.g. notes, alerts reminders etc. you can create them like this

````md
```{note}
Here is a note!
```
````

```{note}
Here is a note!
```

````md
```{warning}
Careful now
```
````

```{warning}
Careful now
```

````md
```{Important}
Read this for sure.
```
````

```{Important}
Read this for sure.
```
